# Slackクローン（ダミーデータ版）

## 目的

Slackの全画面をダミーデータで構築し、2カラムレイアウト + レスポンシブの基礎を習得する。

## 技術スタック

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- react-router-dom

## 画面構成

- **サイドバー**: 幅260px / 背景色 紫 `#611f69` / チャンネル5つを静的に表示
- **ヘッダー**: 現在のチャンネル名を表示
- **メッセージ一覧**: ダミーメッセージ10件
- **入力欄**: 画面下部に固定

## データ

すべてダミーデータ（`src/data/messages.ts`）。DB接続・認証はまだ行わない（**Stage 14以降**で導入予定）。

## レスポンシブ

375px幅ではサイドバーをハンバーガーメニュー + shadcn の `Sheet` に切り替える。
